Liens utiles

Voici une collection de liens qui pourraient vous être utiles lors de l'apprentissage de three.js.
Si vous trouvez quelque chose que vous aimeriez ajouter ici, ou si vous pensez que l'un des liens ci-dessous n'est plus pertinent ou ne fonctionne plus, n'hésitez pas à cliquer sur le bouton 'éditer' en bas à droite et à apporter des modifications !

Notez également que comme three.js est en développement rapide, beaucoup de ces liens contiendront des informations qui sont datées - si quelque chose ne fonctionne pas comme vous l'attendiez ou comme l'un de ces liens le dit, vérifiez la console du navigateur pour les avertissements ou les erreurs. Vérifiez également les pages de documentation pertinentes.

Forums d'aide

Three.js utilise officiellement le [link:https://discourse.threejs.org/ forum] et [link:http://stackoverflow.com/tags/three.js/info Stack Overflow] pour les demandes d'aide. Si vous avez besoin d'assistance pour quelque chose, c'est l'endroit où aller. NE PAS ouvrir de problème sur Github pour les demandes d'aide.
