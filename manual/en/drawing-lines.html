Dessiner des Lignes

Disons que vous voulez dessiner une ligne ou un cercle, pas un `Mesh` en fil de fer. Nous devons d'abord configurer le renderer, la scène et la caméra (voir la page Créer une scène).

Voici le code que nous allons utiliser :

const renderer = new THREE.WebGLRenderer();
renderer.setSize( window.innerWidth, window.innerHeight );
document.body.appendChild( renderer.domElement );

const camera = new THREE.PerspectiveCamera( 45, window.innerWidth / window.innerHeight, 1, 500 );
camera.position.set( 0, 0, 100 );
camera.lookAt( 0, 0, 0 );

const scene = new THREE.Scene();

Ensuite, nous allons définir un matériau. Pour les lignes, nous devons utiliser `LineBasicMaterial` ou `LineDashedMaterial`.

//créer un LineBasicMaterial bleu
const material = new THREE.LineBasicMaterial( { color: 0x0000ff } );

Après le matériau, nous aurons besoin d'une géométrie avec des sommets : 

const points = [];
points.push( new THREE.Vector3( - 10, 0, 0 ) );
points.push( new THREE.Vector3( 0, 10, 0 ) );
points.push( new THREE.Vector3( 10, 0, 0 ) );

const geometry = new THREE.BufferGeometry().setFromPoints( points );

Notez que les lignes sont tracées entre chaque paire de sommets consécutifs, mais pas entre le premier et le dernier (la ligne n'est pas fermée.)

Maintenant que nous avons des points pour deux lignes et un matériau, nous pouvons les assembler pour former une ligne.

const line = new THREE.Line( geometry, material );

Il ne reste plus qu'à l'ajouter à la scène et à appeler `renderer.render()`.

scene.add( line );
renderer.render( scene, camera );

Vous devriez maintenant voir une flèche pointant vers le haut, composée de deux lignes bleues.
