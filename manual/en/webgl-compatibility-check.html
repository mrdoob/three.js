Vérification de compatibilité WebGL

Bien que cela pose de moins en moins de problèmes, certains appareils ou navigateurs peuvent encore ne pas prendre en charge WebGL 2. La méthode suivante vous permet de vérifier s'il est pris en charge et d'afficher un message à l'utilisateur si ce n'est pas le cas. Importez le module de détection de prise en charge WebGL et exécutez ce qui suit avant de tenter de rendre quoi que ce soit. 

import WebGL from 'three/addons/capabilities/WebGL.js';

if ( WebGL.isWebGL2Available() ) {

  // Initialiser la fonction ou d'autres initialisations ici
  animate();

} else {

  const warning = WebGL.getWebGL2ErrorMessage();
  document.getElementById( 'container' ).appendChild( warning );

}
