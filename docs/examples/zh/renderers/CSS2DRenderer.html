CSS 2D渲染器（[name]）

[name]是[page:CSS3DRenderer]（CSS 3D渲染器）的简化版本，唯一支持的变换是位移。

如果你希望将三维物体和基于HTML的标签相结合，则这一渲染器将十分有用。在这里，各个DOM元素也被包含到一个*CSS2DObject*实例中，并被添加到场景图中。
