[name]

Create a smooth 2d spline curve from a series of points. Internally this uses [page:CurveUtils.interpolate] to create the curve.

Example

// Create a sine-like wave var curve = new THREE.SplineCurve( [ new THREE.Vector2( -10, 0 ), new THREE.Vector2( -5, 5 ), new THREE.Vector2( 0, 0 ), new THREE.Vector2( 5, -5 ), new THREE.Vector2( 10, 0 ) ] ); var path = new THREE.Path( curve.getPoints( 50 ) ); var geometry = path.createPointsGeometry( 50 ); var material = new THREE.LineBasicMaterial( { color : 0xff0000 } ); // Create the final object to add to the scene var splineObject = new THREE.Line( geometry, material );
