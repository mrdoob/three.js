[name]

Create a smooth 2d quadratic bezier curve, defined by a startpoint, endpoint and a single control point.

Example

var curve = new THREE.QuadraticBezierCurve( new THREE.Vector2( -10, 0 ), new THREE.Vector2( 20, 15 ), new THREE.Vector2( 10, 0 ) ); var path = new THREE.Path( curve.getPoints( 50 ) ); var geometry = path.createPointsGeometry( 50 ); var material = new THREE.LineBasicMaterial( { color : 0xff0000 } ); //Create the final object to add to the scene var curveObject = new THREE.Line( geometry, material );
