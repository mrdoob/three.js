骨骼（[name]）

骨骼是[page:Skeleton]（骨架）的一部分。骨架是由[page:SkinnedMesh]（蒙皮网格）依次来使用的。 骨骼几乎和空白[page:Object3D]相同。

代码示例

const root = new THREE.Bone(); const child = new THREE.Bone(); root.add( child ); child.position.y = 5;

构造器

[name]( )

创建一个新的[name].

属性

共有属性请参见其基类[page:Object3D]。

[property:Boolean isBone]

Read-only flag to check if a given object is of type [name].

[property:String type]

设置为“Bone”，这可以用于在一个场景中找到所有的Bones。
