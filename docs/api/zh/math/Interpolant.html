插值器（[name]）

A参数样本上插值的抽象基类

参数域是一维的，通常是数据定义的曲线上的时间或路径。

示例值可以具有任何维度，派生类可以对数据应用特殊的解释。

该类提供间隔查找的模板方法，将实际的插值延迟到派生类。

对于最多两个点之间的访问时间复杂度为O(1)，对于随机访问时间复杂度为O(log N)，其中N为位置数。

相关: [link:http://www.oodesign.com/template-method-pattern.html http://www.oodesign.com/template-method-pattern.html]
