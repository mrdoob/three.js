立方纹理（[name]）

创建一个由6张图片所组成的纹理对象。

代码示例

const loader = new THREE.CubeTextureLoader(); loader.setPath( 'textures/cube/pisa/' ); const textureCube = loader.load( [ 'px.png', 'nx.png', 'py.png', 'ny.png', 'pz.png', 'nz.png' ] ); const material = new THREE.MeshBasicMaterial( { color: 0xffffff, envMap: textureCube } );

构造函数

[name]( images, mapping, wrapS, wrapT, magFilter, minFilter, format, type, anisotropy, colorSpace )

CubeTexture（立方贴图）的功能以及用法几乎和[page:Texture]是相同的。区别在于，CubeTexture中的图像是6个单独的图像所组成的数组， 纹理映射选项为[page:Textures THREE.CubeReflectionMapping]（默认值）或[page:Textures THREE.CubeRefractionMapping]。

属性

See the base [page:Texture Texture] class for common properties.

[property:Boolean flipY]

If set to *true*, the texture is flipped along the vertical axis when uploaded to the GPU. Default is *false*.

[property:Boolean isCubeTexture]

Read-only flag to check if a given object is of type [name].

方法

See the base [page:Texture Texture] class for common methods.
