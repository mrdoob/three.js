[name]

字符串类型的关键帧轨道

构造函数

[name]( [param:String name], [param:Array times], [param:Array values] )

[page:String name] - (必须) 关键帧轨道(KeyframeTrack)的标识符.
[page:Array times] - (必须) 关键帧的时间数组.
[page:Array values] - 与时间数组中的时间点对应的值数组.
[page:Constant interpolation] - 使用的插值类型。 取值参考 [page:Animation Animation Constants] 默认值为 [page:Animation InterpolateDiscrete].

属性

参见 [page:KeyframeTrack] 查看继承的属性.

[property:Constant DefaultInterpolation]

默认的插值类型。 参见 [page:Animation InterpolateDiscrete].

[property:Array ValueBufferType]

一个基本数组 (不是 Float32Array 类型, 与 [page:KeyframeTrack] 内的 *ValueBufferType* 属性不一样).

[property:String ValueTypeName]

String 'string'.
