[name]

四元数类型的关键帧轨道。

构造函数

[name]( [param:String name], [param:Array times], [param:Array values] )

[page:String name] - (必须) 关键帧轨道(KeyframeTrack)的标识符.
[page:Array times] - (必须) 关键帧的时间数组.
[page:Array values] - 与时间数组中的时间点对应的值数组.
[page:Constant interpolation] - 使用的插值类型。 取值参考 [page:Animation Animation Constants] 默认值为 [page:Animation InterpolateLinear].

属性

参见 [page:KeyframeTrack] 查看继承的属性.

[property:Constant DefaultInterpolation]

默认的插值类型。 参见 [page:Animation InterpolateDiscrete].

[property:String ValueTypeName]

String 'quaternion'.

方法

参见 [page:KeyframeTrack] 查看继承的方法.

[method:null InterpolantFactoryMethodLinear]()

根据值数组 ([page:KeyframeTrack.values values])， 时间 ([page:KeyframeTrack.times times]) 和值大小 [page:KeyframeTrack.valueSize valueSize] 创建一个新的线性插值 ([page:QuaternionLinearInterpolant QuaternionLinearInterpolant])。
