[name]

光源的基类 - 所有其他的光类型都继承了该类描述的属性和方法。

构造器（Constructor）

[name]( [param:Integer color], [param:float intensity] )

[page:Integer color] - (可选参数) 16进制表示光的颜色。 缺省值 0xffffff (白色)。
[page:Float intensity] - (可选参数) 光照强度。 缺省值 1。

创造一个新的光源。注意，这并不是直接调用的（而是使用派生类之一）。

属性（Properties）

公共属性请查看基类[page:Object3D Object3D]。

[property:Color color]

光源的颜色。如果构造的时候没有传递，默认会创建一个新的 [page:Color] 并设置为白色。

[property:Float intensity]

光照的强度，或者说能量。 在 [page:WebGLRenderer.physicallyCorrectLights physically correct] 模式下, [page:.color color] 和强度 的乘积被解析为以坎德拉（candela）为单位的发光强度。 默认值 - *1.0*

[property:Boolean isLight]

用来校验这个类或者派生类是不是平行光。默认是 *true*。

不应该去改变这个变量，因为内部使用这个变量做了些优化的工作。

Methods

公共方法请查看基类 [page:Object3D Object3D]。

[method:Light copy]( [param:Light source] )

从[page:Light source]复制 [page:.color color], [page:.intensity intensity] 的值到当前光源对象中。

[method:Object toJSON]( [param:object meta] )

以JSON格式返回光数据。

meta -- 包含有元数据的对象，例如该对象的材质、纹理或图片。 将该light对象转换为 three.js [link:https://github.com/mrdoob/three.js/wiki/JSON-Object-Scene-format-4 JSON Object/Scene format]（three.js JSON 物体/场景格式）。
