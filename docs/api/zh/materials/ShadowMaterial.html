阴影材质([name])

此材质可以接收阴影，但在其他方面完全透明。

代码示例

var planeGeometry = new THREE.PlaneBufferGeometry( 2000, 2000 ); planeGeometry.rotateX( - Math.PI / 2 ); var planeMaterial = new THREE.ShadowMaterial(); planeMaterial.opacity = 0.2; var plane = new THREE.Mesh( planeGeometry, planeMaterial ); plane.position.y = -200; plane.receiveShadow = true; scene.add( plane );
