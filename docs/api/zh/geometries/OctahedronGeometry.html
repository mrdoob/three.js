八面几何体（[name]）

一个用于创建八面体的类。

构造器

[name]([param:Float radius], [param:Integer detail])

radius — 八面体的半径，默认值为1。
detail — 默认值为0，将这个值设为一个大于0的数将会为它增加一些顶点，使其不再是一个八面体。

属性

共有属性请参见其基类[page:Geometry]。

[property:Object parameters]

一个包含着构造函数中每个参数的对象。在对象实例化之后，对该属性的任何修改都不会改变这个几何体。
