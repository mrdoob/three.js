网格几何体（[name]）

这个类可以被用作一个辅助物体，来对一个[page:Geometry Geometry]以线框的形式进行查看。

代码示例

const geometry = new THREE.SphereBufferGeometry( 100, 100, 100 ); const wireframe = new THREE.WireframeGeometry( geometry ); const line = new THREE.LineSegments( wireframe ); line.material.depthTest = false; line.material.opacity = 0.25; line.material.transparent = true; scene.add( line );
