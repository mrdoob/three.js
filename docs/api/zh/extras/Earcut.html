[name]

切割多边形三角剖分算法的实现。这份代码是[link:https://github.com/mapbox/earcut mapbox/earcut]的一个端口。

方法

[method:Array triangulate]( data, holeIndices, dim )

data -- 一个顶点坐标的平面数组。
holeIndices -- 空洞索引的数组（如果有的话）。
dim -- 输入数组中每个顶点的坐标数。

通过返回三角形数组对给定的形状定义进行三角剖分。三角形由三个连续的整数定义，这些整数表示顶点索引。
