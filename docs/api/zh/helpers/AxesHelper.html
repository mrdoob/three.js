[name]

用于简单模拟3个坐标轴的对象.
红色代表 X 轴. 绿色代表 Y 轴. 蓝色代表 Z 轴.

代码示例

const axesHelper = new THREE.AxesHelper( 5 ); scene.add( axesHelper );

[page:Number size] -- (可选的) 表示代表轴的线段长度. 默认为 `1`.
