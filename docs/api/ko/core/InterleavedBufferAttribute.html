[name]( [param:InterleavedBuffer interleavedBuffer], [param:Integer itemSize], [param:Integer offset], [param:Boolean normalized] )

프로퍼티

[property:InterleavedBuffer data]

생성자로 전달된 [page:InterleavedBuffer InterleavedBuffer] 인스턴스입니다.

[property:TypedArray array]

[page:InterleavedBufferAttribute.data data].array의 값입니다.

[property:Integer count]

[page:InterleavedBufferAttribute.data data].count의 값입니다. 버퍼가 3-컴포넌트 항목(위치, 법선, 색상 등의)을 저장하고 있을 때, 저장된 항목들의 수를 카운팅합니다.

[property:Boolean isInterleavedBufferAttribute]

Read-only flag to check if a given object is of type [name].

[property:Integer itemSize]

각 항목을 구성하는 값의 수입니다.

[property:String name]

이 속성 인스턴스의 임시 이름입니다. 기본값은 빈 문자열입니다.

[property:Boolean needsUpdate]

기본값은 *false*입니다. *true*로 설정하면 모든 인터리브 버퍼(특정 속성 데이터만이 아닌)를 GPU에 다시 전송합니다.

[property:Boolean normalized]

기본값은 *false*입니다.

[property:Integer offset]

항목이 시작하는 기저 배열 버퍼의 오프셋입니다.
