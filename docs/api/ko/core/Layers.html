[name]

[page:Layers] 오브젝트는 [page:Object3D]를 1개에서 32개의 레이어에 0에서 31의 숫자로 할당합니다. - 내부적으로 레이어들은 [link:https://en.wikipedia.org/wiki/Mask_(computing) bit mask]로 저장되며, 기본값으로 모든 Object3D들은 0번 레이어의 멤버입니다.

가시성을 컨트롤하는데에 사용할 수 있습니다. - 카메라의 뷰가 렌더링될 때 오브젝트는 반드시 [page:Camera camera]와 가시 레이어를 공유해야합니다.

[page:Object3D]로부터 상속받은 모든 클래스들은 [page:Object3D.layers] 프로퍼티를 가지고 있으며 이는 이 클래스의 인스턴스입니다.

예제

[example:webgl_layers WebGL / layers]

생성자

[name]()

새 레이어 오브젝트를 만들고 최초 멤버 레이어 0을 설정합니다.

프로퍼티

[property:Integer mask]

현재 레이어가 속해있는 32 레이어들의 비트 마스크를 저장하고 있습니다.

메서드

[method:undefined disable]( [param:Integer layer] )

layer - 0부터 31까지의 정수.

이 *layer*의 멤버 속성을 제거합니다.

[method:undefined enable]( [param:Integer layer] )

layer - 0부터 31까지의 정수.

이 *layer*에 멤버 속성을 추가합니다.

[method:undefined set]( [param:Integer layer] )

layer - 0부터 31까지의 정수.

*layer*의 멤버 속성을 설정하고, 다른 모든 레이어들의 멤버 속성을 제거합니다.

[method:Boolean test]( [param:Layers layers] )

layers - 레이어 오브젝트

전달받은 *layers* 오브젝트가 적어도 1개 이상의 레이어를 가지고 있으면 true를 리턴합니다.

[method:Boolean isEnabled]( [param:Integer layer] )

layer - an integer from 0 to 31.

Returns true if the given layer is enabled.

[method:undefined toggle]( [param:Integer layer] )

layer - 0부터 31까지의 정수.

ㄴ*layer*의 멤버 속성을 전환합니다.

[method:undefined enableAll]()

모든 레이어에 멤버 속성을 추가합니다.

[method:undefined disableAll]()

모든 레이어의 멤버 속성을 제거합니다.
