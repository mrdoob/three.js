[name]

JSON 포맷으로 로드된 폰트를 표현하는 [page:Shape Shapes] 세트를 만듭니다.

[page:FontLoader]에서 내부적으로 사용됩니다.

예제

[example:webgl_geometry_text_shapes geometry / text / shapes ]

생성자

[name]( data )

data -- 폰트를 표현하는 JSON 데이터입니다.

이 생성자는 [page:Shape Shapes] 배열의 새 [name]를 만듭니다.

프로퍼티

[property:Array data]

생성자에 전달되는 JSON 데이터입니다.

메서드

[method:null generateShapes]( [param:String text], [param:Float size] )

[page:String text] -- 텍스트 문자열입니다.
[page:Float size] -- (생략가능) [page:Shape Shapes]의 크기입니다. 기본값은 *100*입니다.
폰트에서 텍스트를 표현하는 [page:Shape Shapes] 배열을 만듭니다.
