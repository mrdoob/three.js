[name]

[name]은 미리 설정해놓은 카메라로 장면을 효율적으로 렌더릴할 때 사용될 수 있습니다. VR 장면을 렌더링할 때 중요한 퍼포먼스적 측면이기도 합니다.
[name] 인스턴스는 항상 하위 카메라들의 배열을 가지고 있습니다. 각 하위 카메라에는 이 카메라로 렌더링되는 뷰포트 부분을 결정하는 *뷰포트* 속성을 반드시 정의해야 합니다.

예제

[example:webgl_camera_array camera / array ]

생성자

[name]( [param:Array array] )

카메라 배열.

프로퍼티

일반 프로퍼티는 기본 [page:PerspectiveCamera]를 참고하세요.

[property:Array cameras]

카메라 배열.

[property:Boolean isArrayCamera]

Read-only flag to check if a given object is of type [name].
