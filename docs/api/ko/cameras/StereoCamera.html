듀얼 [page:PerspectiveCamera PerspectiveCamera]로서 [link:https://en.wikipedia.org/wiki/Anaglyph_3D 3D Anaglyph] 혹은 [link:https://en.wikipedia.org/wiki/parallax_barrier Parallax Barrier] 효과에 사용됩니다.

생성자

[name]( )

프로퍼티

[property:Float aspect]

기본값은 *1*입니다.

[property:Float eyeSep]

기본값은 *0.064*입니다.

[property:PerspectiveCamera cameraL]

왼쪽 카메라입니다. [page:Layers layer 1]에 추가됩니다. - 왼쪽 카메라에 렌더링되는 객체들은 이 레이어에도 추가되어야 합니다.

[property:PerspectiveCamera cameraR]

오른쪽 카메라입니다. [page:Layers layer 2]에 추가됩니다. - 오른쪽 카메라에 렌더링되는 객체들은 이 레이어에도 추가되어야 합니다.

메서드

[method:undefined update]( [param:PerspectiveCamera camera] )

전달받은 카메라에 기반해 스테레오 카메라를 업데이트합니다.
