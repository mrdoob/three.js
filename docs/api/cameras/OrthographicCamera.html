[name]

是一种使用 [link:https://en.wikipedia.org/wiki/Orthographic_projection orthographic projection] 的相机。

在此投影模式下，渲染图像中对象的大小保持不变 而不管其与相机的距离如何。

这对于渲染 2D 场景和 UI 元素尤其有用。

构造函数

[name]( [param:Number left], [param:Number right], [param:Number top], [param:Number bottom], [param:Number near], [param:Number far] )

left — 相机截头锥体可视空间（Camera frustum）左平面。
right — 相机截头锥体可视空间右平面。
top — 相机截头锥体可视空间上平面。
bottom — 相机截头锥体可视空间下平面。
near — 最近的相机截头锥体可视空间平面的距离。
far — 最远的相机截头锥体可视空间平面的距离。

它们一起定义了相机的 [link:https://en.wikipedia.org/wiki/Viewing_frustum viewing frustum].

属性

通过 [page:Camera] 基类，可了解相关公共属性。
请注意，在对大多数属性进行更改后，你必须调用 [page:OrthographicCamera.updateProjectionMatrix .updateProjectionMatrix] 以使更改生效。

[property:Float bottom]

相机截头锥体可视空间的下平面。

[property:Float far]

最远相机截头锥体可视空间平面的距离。默认值为 *2000*.

有效值范围介于当前 [page:.near near] 平面的 near 值到无穷远。

[property:Boolean isOrthographicCamera]

用于测试此类是否是 OrthographicCameras（正投影相机）或派生于 OrthographicCameras。默认值是 *true*。

不要更改它，因为渲染器内部要使用它进行优化操作。

[property:Float left]

相机截头锥体可视空间左平面

[property:Float near]

最近相机截头锥体可视空间平面的距离。默认值是 *0.1*。

有效值范围介于 0 到当前 [page:.far far] 平面的距离。 请注意，与 [page:PerspectiveCamera] 不同的是，OrthographicCamera 的 near 值可以为 *0*，它是一个合法有效值。

[property:Float right]

相机截头锥体可视空间右平面。

[property:Float top]

相机截头锥体可视空间上平面。

[property:Object view]

通过 [page:OrthographicCamera.setViewOffset setViewOffset] 来设置。默认值是 *null*。

[property:number zoom]

获取或设置相机的缩放值。默认值是 *1*。

方法

通过 [page:Camera] 基类，可了解相关公共方法。

[method:null setViewOffset]( [param:Float fullWidth], [param:Float fullHeight], [param:Float x], [param:Float y], [param:Float width], [param:Float height] )

fullWidth — 多视角（multiview）设置的全宽度
fullHeight — 多视角设置的全高度
x — 辅助相机（subcamera）水平偏移量
y — 辅助相机垂直偏移量
width — 辅助相机宽度
height — 辅助相机高度

在一个较大的 [link:https://en.wikipedia.org/wiki/Viewing_frustum viewing frustum] 中设置偏移量。 这对于多窗口或多显示器/多机器设置会很有用。 有关如何使用它的示例，请参见 [page:PerspectiveCamera.setViewOffset PerspectiveCamera]。

[method:null clearViewOffset]()

清除 .setViewOffset 方法设置的所有偏移量。

[method:null updateProjectionMatrix]()

更新相机投影矩阵。必须在相关参数都更改完成后再调用。

[method:JSON toJSON]()

以JSON格式返回相机的数据。
