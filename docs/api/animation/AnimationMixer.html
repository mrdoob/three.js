[name]

AnimationMixer 是场景中某个特定对象的动画播放器，当场景中的多个对象单独动画的时候，每个对象都可以使用一个 AnimationMixer。

有关 three.js 动画系统的不同元素概览，请参阅本手册 'Next Steps' 部分的 'Animation System' 一文查看。

构造器

[name]( [param:Object3D rootObject] )

[page:Object3D rootObject] —— 由这一 mixer 播放动画的对象。

Properties

[property:Number time]

全局 mixer time（以秒为单位，以 mixer 的创建为开始，开始时间为0）。

[property:Number timeScale]

全局缩放因子 [page:.time mixer time]。

注意：设置 mixer 的 timeScale 为 0，然后再设置为 1 可能会让该 mixer 控制的所有 action 暂停/取消暂停。

方法

[method:AnimationAction clipAction]([param:AnimationClip clip], [param:Object3D optionalRoot])

为传入的 clip 返回一个 [page:AnimationAction]，可以选择性地使用一个不同于该 mixer 默认的根对象的根对象。 第一个参数可以是一个 [page:AnimationClip] 对象或一个 AnimationClip 的名称。

如果不存在满足这些参数的 action，这个方法会新建一个。

注意：多次调用该方法并传入相同的参数将始终返回同一个 clip 实例。

[method:AnimationAction existingAction]([param:AnimationClip clip], [param:Object3D optionalRoot])

为传入的 clip 返回存在的 [page:AnimationAction]，可以选择性地使用一个不同于该 mixer 默认的根对象的根对象。

第一个参数可以是一个 [page:AnimationClip] 对象或一个 AnimationClip 的名称。

[method:Object3D getRoot]()

返回这一 mixer 的根对象。

[method:AnimationMixer stopAllAction]()

关闭该 mixer 上之前计划好的所有 action。

[method:AnimationMixer update]([param:Number deltaTimeInSeconds])

增加全局 mixer time 并更新动画。

通常在渲染循环中，通过 mixer 的 [page:.timeScale timeScale] 缩放传入的 [page:Clock.getDelta clock.getDelta] 来完成这一操作。

[method:null uncacheClip]([param:AnimationClip clip])

为传入的 clip 释放所有的内存资源。

[method:null uncacheRoot]([param:Object3D root])

为传入的根对象释放所有的内存资源。

[method:null uncacheAction]([param:AnimationClip clip], [param:Object3D optionalRoot])

为 action 释放所有的内存资源。
