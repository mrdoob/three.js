[name]

一个 AnimationClip （动画裁剪）是一组可重用的关键帧跟踪（key frame track），可用来表示一个动画。

有关 three.js 动画系统的不同元素概览，请参阅本手册 'Next Steps' 部分的 'Animation System' 一文查看。

构造器

[name]( [param:String name], [param:Number duration], [param:Array tracks] )

[page:String name] —— 该 clip 的名称。
[page:Number duration] —— 该 clip 的持续时间（单位为秒）。 如果传入的值为负，持续时间将会从传入的 *tracks* 数组计算。
[page:Array tracks] —— 一个 [page:KeyframeTrack KeyframeTracks] 数组。

注意：除了用构造器直接实例化一个 AnimationClip，也可以用它的一些静态方法创建 AnimationClip： JSON（[page:.parse parse]），变形目标序列（[page:.CreateFromMorphTargetSequence CreateFromMorphTargetSequence]，[page:.CreateClipsFromMorphTargetSequences CreateClipsFromMorphTargetSequences]）或动画层次（animation hierarchies，[page:.parseAnimation parseAnimation]）—— 如果你的模型还没有将 AnimationClip 保存在它的 geometry 的动画数组中。

属性

[property:Number duration]

这一 clip 的持续时间（以秒为单位）。可以通过 [page:.resetDuration resetDuration] 从 [page:.tracks tracks] 数组计算得到。

[property:String name]

这一 clip 的名称。可以通过 [page:.findByName findByName] 搜索到某个 clip。

[property:Array tracks]

一个数组，包含通过这一 clip 动画的每个属性的 [page:KeyframeTrack]。

[property:String uuid]

这一 clip 实例的 [link:http://en.wikipedia.org/wiki/Universally_unique_identifier UUID]。 是被自动分配的，不应该编辑该属性。

方法

[method:AnimationClip optimize]()

通过删除等效的序列键（equivalent sequential keys，在变形目标序列中很常见）来优化每一个 track（上文的关键帧跟踪）。

[method:null resetDuration]()

将 clip 的 [page:.duration duration] 设置为它的最长 [page:KeyframeTrack] 的持续时间。

[method:AnimationClip trim]()

按照该 clip 的持续时间分割所有 track 。

静态方法

[method:Array CreateClipsFromMorphTargetSequences]( [param:String name], [param:Array morphTargetSequence], [param:Number fps], [param:Boolean noLoop] )

返回一个由传入的 geometry 的 [page:Geometry.morphTargets morph target sequences] 生成的新 AnimationClip 数组， 并试图将变形目标基于动画组模式分类，例如 "Walk_001, Walk_002, Run_001, Run_002 ..."

这一方法通过 [page:JSONLoader] 内部调用，并且使用 [page:.CreateFromMorphTargetSequence CreateFromMorphTargetSequence]。

[method:AnimationClip CreateFromMorphTargetSequence]( [param:String name], [param:Array morphTargetSequence], [param:Number fps], [param:Boolean noLoop] )

由传入的 geometry 的 [page:Geometry.morphTargets morph targets array]，返回一个新的 AnimationClip， 需要传入名称和帧率。

注意：fps 参数是必需的，但在 *AnimationAction*中， 动画速度可以通过 [page:AnimationAction.setDuration animationAction.setDuration] 被覆盖。

[method:AnimationClip findByName]( [param:Object objectOrClipArray], [param:String name] )

通过名称搜索 AnimationClip，第一个参数看做是包含了一个名为 "animation" 数组的 AnimationClip 数组，或 mesh 或 geometry。

[method:AnimationClip parse]( [param:Object json] )

解析一个 JSON 格式的 clip 并返回一个 AnimationClip。

[method:AnimationClip parseAnimation]( [param:Object animation], [param:Array bones] )

解析 animation.hierarchy 格式并返回一个 AnimationClip。

[method:Object toJSON]( [param:AnimationClip clip] )

接收一个 AnimationClip，并返回其对应的 JSON 对象。
