[name]

تتبع للقيمة الرقمية الخاصة بـإطار رباعي *quaternion keyframe*.

المنشئ (Constructor)

[name]( [param:String name], [param:Array times], [param:Array values] )

[page:String name] (ضرورية) المعرف الخاص بالـ*KeyframeTrack*.
[page:Array times] - (ضرورية) جدول الأوقات الخاصة بـ*keyframe*.
[page:Array values] - القيم الخاصة بالـ*keyframes* في وقت محدد.
[page:Constant interpolation] - نوع الـ*interpolation* الذي يجب إستعماله. أنظر [page:Animation Animation Constants] من أجل القيم المتاحة. الإفتراضي هو [page:Animation InterpolateLinear].

الخصائص (Properties)

أنظر [page:KeyframeTrack] من أجل الخصائص الموروثة

[property:Constant DefaultInterpolation]

نوع الـ*interpolation* الافتراضي المراد استخدامه ،

[property:String ValueTypeName]

String 'quaternion'.
