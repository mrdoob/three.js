المنشئ (Constructor)

[name]( [param:String name], [param:Array times], [param:Array values] )

[page:String name] - (ضرورية) معرّف KeyframeTrack.
[page:Array times] - (ضرورية) جدول الأوقات الخاصة بـ*keyframe*.
[page:Array values] - القيم الخاصة بالـ*keyframes* في وقت محدد.
[page:Constant interpolation] - نوع الـ*interpolation* الذي يجب إستعماله. أنظر [page:Animation Animation Constants] من أجل القيم المتاحة. الإفتراضي هو [page:Animation InterpolateDiscrete].
