Polyfills

three.js 包含 polyfills 的下列函数和常量。

[page:Number.EPSILON Number.EPSILON]

1 和大于 1 的最小值之间的差值，这个差值可以表示为一个数字。 [link:https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Number/EPSILON MDN reference].

[page:Math.sign Math.sign]( [page:Number x] )

如果参数是正数，负数，正零或负零， 函数会相应地返回 1，-1，0 或者 -0。否则，返回 NaN。 [link:https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Math/sign MDN reference].

[page:Function.prototype.name Function.prototype.name]( [page:Number x] )

返回一个函数的名称，或者（ES6 之前的实现）空字符串（，如果是）匿名函数（的话）。 [link:https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Function/name MDN reference].

[page:Object.assign Object.assign]( [page:Object target], [page:Object ...sources] )

Object.assign() 方法用于将所有可枚举的自身属性的值从一个或多个源对象复制到目标对象。 它会返回目标对象。 [link:https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Object/assign MDN reference].
