在示例中为什么会有 meta 标签？

<meta name="viewport" content="width=device-width, user-scalable=no, minimum-scale=1.0, maximum-scale=1.0">

这些标签控制了 viewport 在移动端浏览器中的大小和缩放比例（这些浏览器中，页面内容可能会以不同于可见 viewport 的大小呈现）。

怎样在调整大小时保持场景比例？

我们希望所有对象，无论它们与相机之间的距离如何，都会显示相同的大小，即使窗口大小已调整。 解决这个问题的关键方程是计算给定距离处可见高度的公式： visible_height = 2 * Math.tan( ( Math.PI / 180 ) * camera.fov / 2 ) * distance_from_camera; 如果我们将窗口高度增加一定的百分比，那么我们想要的是所有距离处的可见高度以相同的百分比增加。 通过改变相机的位置是不能达到这一效果的，除非改变相机的视场。 [link:http://jsfiddle.net/Q4Jpu/ Example].

为什么我对象的一部分不可见？

这可能是因为设置了背景剔除（face culling），面是有方向的，方向决定了哪面是正面哪面是背面。正常情况下，背景剔除会移除（不渲染）背面。 要查看这是否是您的问题，请将 material 的 side 改为 THREE.DoubleSide。 material.side = THREE.DoubleSide
