This guide provides a brief overview of the basic components of a web-based VR application made with three.js.

Workflow

You have to include [link:https://github.com/mrdoob/three.js/blob/master/examples/js/vr/WebVR.js WebVR.js] into your project. *WEBVR.createButton()* does two important things: First, it creates a button which indicates VR compatibility. Second, it initiates a VR session if the user activates the button. The only thing you have to do is to add the following line of code to your app.

document.body.appendChild( WEBVR.createButton( renderer ) );

Next, you have to tell your instance of *WebGLRenderer* to enable VR rendering:

renderer.vr.enabled = true;

Finally, you have to adjust your animation loop since we can't use our well known *window.requestAnimationFrame()* function. The minimal code looks like this:

function animate() { renderer.setAnimationLoop( render ); }
