*Inheritance: EventDispatcher → Texture → Data3DTexture →*

# VOXData3DTexture

A VOX 3D texture.

Instances of this class are created from the loaded chunks of [VOXLoader](VOXLoader.html).

## Constructor

### new VOXData3DTexture( chunk : Object )

Constructs a new VOX 3D texture.

**chunk**

A VOX chunk loaded via [VOXLoader](VOXLoader.html).
