# BlendShader

## Import

BlendShader is an addon, and must be imported explicitly, see [Installation#Addons](https://threejs.org/manual/#en/installation).

```js
import { BlendShader } from 'three/addons/shaders/BlendShader.js';
```

## Properties

### .BlendShader : ShaderMaterial~Shader (inner, constant)

Blends two textures.
