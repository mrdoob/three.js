# DotScreenShader

## Import

DotScreenShader is an addon, and must be imported explicitly, see [Installation#Addons](https://threejs.org/manual/#en/installation).

```js
import { DotScreenShader } from 'three/addons/shaders/DotScreenShader.js';
```

## Properties

### .DotScreenShader : ShaderMaterial~Shader (inner, constant)

Dot screen shader based on [glfx.js sepia shader](https://github.com/evanw/glfx.js).
