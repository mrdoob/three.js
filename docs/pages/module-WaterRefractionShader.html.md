# WaterRefractionShader

## Import

WaterRefractionShader is an addon, and must be imported explicitly, see [Installation#Addons](https://threejs.org/manual/#en/installation).

```js
import { WaterRefractionShader } from 'three/addons/shaders/WaterRefractionShader.js';
```

## Properties

### .WaterRefractionShader : ShaderMaterial~Shader (inner, constant)

Basic water refraction shader.
