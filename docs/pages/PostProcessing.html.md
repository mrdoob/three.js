# PostProcessing

### new PostProcessing( renderer : Renderer, outputNode : Node.<vec4> )

Constructs a new post processing management module.

**renderer**

A reference to the renderer.

**outputNode**

An optional output node.

**Deprecated:** since r183. Use [RenderPipeline](RenderPipeline.html) instead. PostProcessing has been renamed to RenderPipeline. This class is a wrapper for backward compatibility and will be removed in a future version.
