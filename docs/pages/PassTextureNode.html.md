# PassTextureNode

Represents the texture of a pass node.

## Constructor

### new PassTextureNode( passNode : PassNode, texture : Texture )

Constructs a new pass texture node.

**passNode**

The pass node.

**texture**

The output texture.

## Properties

### .passNode : PassNode

A reference to the pass node.
