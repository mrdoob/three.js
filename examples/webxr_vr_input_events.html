three.js vr - input events

This example demonstrates the use of Motion Controllers library to create dynamic WebXR input mappings. Controls for the demo:
Thumbstick and touchpad forward/backward to smoothly slide
and left/right to snap-turn in 45-degree steps
Trigger to pull the cube closer
Squeeze to make the cube smaller
Any of the buttons to randomize the cube's color

See the comments in the code for more information.
