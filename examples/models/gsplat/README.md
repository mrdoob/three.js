The `procedural_flower.ply` asset is a deterministic procedural 3D Gaussian
splat sample generated for the three.js examples. It is released under CC0-1.0.

The `procedural_flower.gltf` asset is a compact, deterministic
`KHR_gaussian_splatting` version generated for the glTF Gaussian splatting
example. It is released under CC0-1.0.
